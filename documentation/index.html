Installation

Prerequisites

Although Reaxy can be run without any development experience, it would be much easier if you already have some. The following instructions allow you to run a local copy on your machine.

Install tools

If you have not yet installed nodejs , please Download and globally install nodejs : https://nodejs.org
Note: download Recommended For Most Users version


A detailed instruction on how to install NodeJS is available here.

After installing NodeJS there is just one thing left to do: Simply install Typings globally by running the command : npm install -g typings and wait for npm to do the dirty work.


Note: Make sure you have Node version >= 18 and NPM >= 16 . Also globally installed typescript.


Installing React-ts globally is as simple as running this simple command : npx create-react-app my-app --template typescript


After the tools is installed, go inside of the template directory and install dependencies:


Run npm install to install node dependencies defined in package.json.


Running local copy


To run a local copy in development mode, execute npm run dev and go to http://localhost:5173 in your browser.


Run npm build to run the local copy in production mode and build the sources. This will build the production version of the application. Codes of all html, css, js and other libraries are minified and placed in the dist folder. You can deploy the contents of this folder to your production server when you publish the application.


This was the downloads considered for the default react app

The React Router DOM can be used to update your app's URL and navigate between different pages by working in sync with the browser history. This way, users can see different content at different URLs and your application becomes more modular. go: https://reactrouter.com

Note
  • Could you please take a look inside the file src/layout/rightSidebar/content/mode/content/Content.tsx ? Because it contains a forEach loop on an array, any values you add to the myPrimaryMode array in constants.ts will automatically become part of your selections. However, it's important to note that adding objects to the array should be in accordance with the required structure.
                
                  
import React from "react";
import { useAppDispatch } from "@config/hooks";
import { Stack } from "@mui/material";
import { setPrimaryMode } from "@config/redux";
import { myPrimaryMode } from "@config/constant";
import SkinApp from "@config/assets/images/rightSidebar/SkinApp";
import { styles } from "./Content.styles";

export function Content() {
  // ** dispatch
  const dispatch = useAppDispatch();

  return (
      <Stack  sx={styles.container} spacing={0.5} pt={2} direction={"row"}>
         {Object.values(myPrimaryMode).map((mode) => (
              <Stack spacing={1} key={mode.value}>
              <SkinApp
                  onClick={() => dispatch(setPrimaryMode(mode.value))}
                  width={"120px"}
                  height={"100px"}
                  color1={mode.primary}
                  color2={mode.default}
                  />
         ))}
         </Stack>
      );
    }

Create New Page

Reaxy uses React router dom for navigation.


For more information on React component structure React component


To create a new page , do the following:


1. Create new page 
                  
                    import React from 'react'
                    import { Box } from "@mui/material"

                    export default function NewPage() {
                      return (
                    <Box>
                          Hello world
                    <Box>
                      )
                    }

2. Announce a route 
                  
                    const NewPage = lazy(() => import("@app/pages/NewPage"));


                    export const router = createBrowserRouter(
                      createRoutesFromElements(
                        <Route path="/new-page" element={  <NewPage />} />
                      )
